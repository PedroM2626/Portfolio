# Portfolio Pessoal

[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](https://opensource.org/licenses/MIT)

Bem-vindo ao meu portfólio pessoal! Este é um projeto desenvolvido para mostrar meus trabalhos, habilidades e experiências de forma profissional e interativa.

## 🚀 Tecnologias

- [React](https://reactjs.org/)
- [TypeScript](https://www.typescriptlang.org/)
- [Vite](https://vitejs.dev/)
- [Tailwind CSS](https://tailwindcss.com/)

## ✨ Recursos

- Design responsivo
- Navegação suave
- Seção de projetos
- Formulário de contato
- Tema claro/escuro
- SEO otimizado

## 🛠️ Como executar localmente

1. Clone o repositório:
   ```bash
   git clone https://github.com/PedroM2626/Portfolio.git
   cd portfolio
   ```

2. Instale as dependências:
   ```bash
   npm install
   # ou
   yarn
   ```

3. Inicie o servidor de desenvolvimento:
   ```bash
   npm run dev
   # ou
   yarn dev
   ```

4. Abra [http://localhost:5173](http://localhost:5173) no seu navegador.

## 📦 Construção

Para criar uma versão de produção:

```bash
npm run build
# ou
yarn build
```
